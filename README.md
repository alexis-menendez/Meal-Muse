##  Getting Started
About
Meal Muse is a meal planning website to help combat meal planning fatigue. We offer  month to month planning, a recipe inventory and helpful to links to discover more! 

### Prerequisites

1. Working internet
2. Hunger
3. the ability to tolerate cooking
